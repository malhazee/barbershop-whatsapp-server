# WhatsApp Server - نظام إشعارات واتساب لصالون الحلاقة

نظام إرسال رسائل واتساب تلقائية عند:
- حجز موعد جديد
- إلغاء موعد
- إتمام الخدمة

## التشغيل

```bash
npm install
npm start
```

## المتغيرات المطلوبة

- `PORT`: منفذ السيرفر (افتراضي: 3000)

## الاستخدام

بعد التشغيل، افتح المتصفح على العنوان المعروض لمسح QR Code وربط واتساب.
